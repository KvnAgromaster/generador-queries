Instrucciones Instalacion
1. Instalar node "https://nodejs.org/es/download", puede descargar la version de windows installer .msi
2. Abrir el proyecto y agregar el archivo .env
3. Revisar las credenciales del archivo .env
4. Hacer "npm i" en la consola de la carpeta del proyecto, esto instalara todas las dependencias que necesita para funcionar automaticamente
5. Para correr el programa usar "node obtenerQueries.js"
6. Revisar archivo generado en querys
7. De nada xd
